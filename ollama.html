Information

This chat interface is designed to work with Ollama, a platform for running large language models.

To use this interface:

  1. Make sure your Ollama server is running
  2. Configure the endpoint and model name in the settings
  3. Start chatting with your model

Popular Ollama models:

  • llama3.2 - Meta's latest Llama 3.2 model
  • llama3 - Meta's Llama 3 model
  • mistral - Mistral AI's model
  • codellama - Code-specific Llama model
  • phi3 - Microsoft's lightweight model
  • gemma - Google's Gemma model
  • tinydolphin - Your current model
