Connection Troubleshooting

If you're having connection issues:

  • Make sure Ollama is running on your machine
  • Verify the endpoint URL is correct
  • Check that the model name is available (run ollama list)
  • Try pulling the model: ollama pull llama3
  • If using a different port, update the endpoint accordingly

This chat interface connects to your local Ollama instance. Make sure Ollama is running on your machine.

To install and run Ollama:

  1. Download from https://ollama.ai
  2. Install and run the application
  3. Pull a model: ollama pull llama3
